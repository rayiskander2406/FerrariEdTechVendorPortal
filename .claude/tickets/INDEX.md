## Filing a New Bug

Use the `/bug` command:
```
/bug [description of the bug]
```

The command will:
1. Ask clarifying questions
2. Perform root cause analysis
3. Create a dev spike
4. Design test plan (100% coverage)
5. Create ticket file
6. Update TODO.md and PLANNING.md
