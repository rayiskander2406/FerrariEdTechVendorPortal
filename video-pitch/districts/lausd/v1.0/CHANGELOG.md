# LAUSD Video Pitch - v1.0 Changelog

## v1.0.0 (December 2, 2025)

### Initial Release

**Duration**: 85.7 seconds | **Resolution**: 1920x1080 | **Size**: 7.5 MB

#### Features
- 7-scene animated explainer video
- Edge TTS voiceover (en-US-AndrewNeural voice)
- Professional motion graphics with Remotion

#### Scenes
1. **Hook** (0:00-0:10) - 670K students, 1000 schools, vendor access challenge
2. **Problem** (0:10-0:23) - Privacy team drowning, 71-question apps, Clever fees
3. **Solution** (0:23-0:37) - PoDS-Lite 13 questions, tokenization, 80% auto-approve
4. **Integration** (0:37-0:54) - SSO, OneRoster, LTI 1.3, tokenized messaging
5. **Benefits** (0:54-1:07) - 80% fewer reviews, auto compliance, no middleman
6. **Leverage** (1:07-1:18) - Vendor value = district leverage for discounts
7. **Close** (1:18-1:26) - Call to action

#### Key Messages
- Privacy-safe tokenization (zero PII exposure)
- 80% faster vendor approvals
- Clever fee elimination ($16-19/school/month)
- Full integration breadth (SSO, Rostering, LTI, Messaging)
- Leverage for vendor discounts

#### Technical Stack
- **Video**: Remotion (React-based)
- **TTS**: Edge TTS (Microsoft Neural)
- **Voice**: en-US-AndrewNeural (Confident, Authentic, Warm)
- **Audio**: Single concatenated track for sync
- **Format**: H.264 + AAC
